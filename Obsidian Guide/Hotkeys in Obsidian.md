| Key Combination            | What It Does                                       |
| -------------------------- | -------------------------------------------------- |
| `Ctrl+P`                   | Command palette — where you can perform any action |
| `Ctrl+O`                   | Quick switcher — search notes by name              |
| `Ctrl+Alt+O`               | Omnisearch — search notes by name **and content**  |
| `Ctrl+E`                   | Edit/preview mode toggle                           |
| `Ctrl+Alt+W`               | Source code mode                                   |
| `Ctrl+;`                   | Add file property (tags)                           |
| `Ctrl+N` or `Ctrl+Shift+N` | Create a note                                      |
| `Ctrl+H`                   | Find and replace words                             |
| `Alt+C`                    | Move current file to another folder                |
| `Ctrl+F`                   | Search within the note                             |
| `Ctrl+D`                   | Delete paragraph                                   |
| `Alt+LMB`                  | Edit in multiple places at once                    |
| `Ctrl+W`                   | Close current tab                                  |

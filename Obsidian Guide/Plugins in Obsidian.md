Don't overload your vault with plugins, or you'll just get more confused! I have pre-installed the bare minimum (in my opinion) for comfortable work and handling everyday tasks. 

If you're looking for something specific, try using search and the internet. No luck? Try asking an AI.

# I Can Recommend
1. [Auto Hide](obsidian://show-plugin?id=obsidian-auto-hide) — hides the left and right panels when clicking on the main note (useful for small monitors).
2. [File Explorer Note Count](obsidian://show-plugin?id=file-explorer-note-count) — counts how many notes are in folders (only counts `md` files).
3. [Hider](obsidian://show-plugin?id=obsidian-hider) — lets you hide unnecessary elements. Play around and hide what you don't need.
4. [Editor Width Slider](obsidian://show-plugin?id=editor-width-slider) — convenient note width adjustment. ^0364c5
5. [Password Protection](obsidian://show-plugin?id=password-protection) — protects the vault from snoops with a password, without real encryption. You can set a timer for how long of inactivity before a password is required. For example, so that after stepping away for three minutes, your kids or spouse can't read your notes. If you forget the password, it's easy to remove by just deleting the plugin at: `\.obsidian\plugins\password-protection`. 
6. [Youglish Plugin](obsidian://show-plugin?id=obsidian-youglish-plugin) — an overpowered language learning plugin. You can select a word (in English), click the plugin button, and automatically get a selection of YouTube videos using that word + pronunciation right inside the vault. ^a8c4b7
7. [Pomodoro Widget](https://github.com/gvvim/obsidian-pomodoro-widget) — a timer/stopwatch. I consider it very useful, since just recently I forgot some boiling eggs for about an hour and a half, and only snapped out of it when they exploded all over my kitchen. Setup is intuitive, the interface is clear, the widget is pretty and easy to hide. Has a bell and ticking sounds you can turn off. The downsides are it's unclear how to change styles and seconds are hard to configure.
8. [Tag Wrangler](obsidian://show-plugin?id=tag-wrangler) — bulk rename tags, for example, rename to any other tag (`Right-click` when the plugin is installed).
9. [Copy Block Link](obsidian://show-plugin?id=obsidian-copy-block-link) — lets you conveniently link to headings or paragraphs using the mouse.
10.  [Quick Explorer](obsidian://show-plugin?id=quick-explorer) — faster folder navigation using the top panel. Supports hotkeys. I have it in my vault to open the left panel less often and reduce confusion.
11. [Media DB](obsidian://show-plugin?id=obsidian-media-db-plugin) — for creating your own database of games, movies, TV shows, wiki articles, books, and more. Automatic metadata (properties) embedding. Lets you unleash your creativity, process information, and capture emotions in the moment. [In this video](https://www.youtube.com/watch?v=4sEMz0UVkJc) the author shows how to set up Media DB. No API required. If the game isn't on Steam, try OMD. And if it's not there either, add the game manually (or create a separate note for such projects). ^e1aab8
12. [Book Search](obsidian://show-plugin?id=obsidian-book-search-plugin) — search for books and automatically embed them in the vault with all metadata (properties). Can be useful for your own database. But requires setup ([the beginning of this video has an explanation](https://youtu.be/NuFeyWUob_Y), however I don't recommend using their scheme — it's impractical. Can be done better with [bases](https://help.obsidian.md/bases)). No API required, but many requests in one day get capped, so filling a library in one sitting won't work. ^b3e7c2
13. [Clear Unused Images](obsidian://show-plugin?id=oz-clear-unused-images) — clears the cache of images that have no hyperlinks. I use it occasionally to clean the vault. But be careful and archive the vault before such global changes. Also it doesn't touch other files, i.e. it won't delete videos and PDFs.
14. [Lazy Plugin Loader](obsidian://show-plugin?id=lazy-plugins) — faster vault startup when there are many plugins or the vault has grown large (I use it). No setup required.
15. [Advanced Tables](obsidian://show-plugin?id=table-editor-obsidian) — simplified table editing for those who like to tinker.
16. [Export Image](obsidian://show-plugin?id=obsidian-export-image) — awesome and simple export of notes as images (you can add an author badge and adjust image width and page split style). The quality is much higher than copying via apps like Windows Snipping Tool, ShareX, or Greenshot.
17. [Telegram Sync](obsidian://show-plugin?id=telegram-sync) — automatically saves Telegram posts to your vault using a bot. Setup is straightforward, [here's a video](https://youtu.be/xV02gHXZULM?t=597). However, it doesn't support formats other than posts — so plain text or images won't be captured. Don't repeat my mistakes and don't save everything indiscriminately; better create a Telegram group for such posts... ^dd3c02
18. [Obsidian Web Clipper](https://obsidian.md/clipper) — turn websites into notes, highlight needed fragments. It's a browser extension. ^222e3e
19. [Big Calendar](https://github.com/Quorafind/Obsidian-Big-Calendar) — a decent alternative to the task calendar. This plugin supports specific assignment times, but can't read tasks from Kanban, so I didn't use it.
20. [Task Genius](obsidian://show-plugin?id=obsidian-task-progress-bar) — a powerful plugin for collecting tasks from the entire vault and managing/tracking them in different views. Has projects, plans, matrices, and more. Overkill for regular users, not very flexible, and conflicts with my kanban boards, so I didn't add it. **Has Russian interface**. They even have [a website](https://taskgenius.md/).
21. [Notebook Navigator](obsidian://show-plugin?id=notebook-navigator) — Tag navigation with automatic thumbnail detection using images inside notes (video links, [[Plugins in Obsidian#^b89a28|website images]], etc.). Fine-grained settings and display modes. Has Russian interface.

# Conclusions
So anyway... as you can see, there are a lot of plugins. They are the heart of Obsidian, coming in big and small. From editing Excel spreadsheets to Bible references. Some particularly useful ones may still be unpopular, so you'll have to find the ones you need manually through the [Obsidian store](https://obsidian.md/plugins), the app, or the internet.
